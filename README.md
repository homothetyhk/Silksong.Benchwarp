# Silksong.Benchwarp

Work-in-progress port of Hollow Knight's [Benchwarp mod](https://github.com/homothetyhk/HollowKnight.BenchwarpMod) to Hollow Knight: Silksong.

## Dependencies

- AutoHookGenPatcher, available on Thunderstore.
